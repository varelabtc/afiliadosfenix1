Links Master

Links base que serão distribuídos para os afiliados

🔗

Como funciona?

Os links master são as URLs base de cada casa de apostas. Quando um afiliado cria um link, o sistema automaticamente adiciona o código único dele à URL. Use as variáveis {AFFILIATE_CODE} e {SUBID} nas URLs.
